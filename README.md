# Project: Hotel Manager

-   **Description**: a smart way to manage your hotel.
-   **Language**: Javascript - React native (Expo).
-   **Flatform**: android, ios.
-   **Dev**: Nguyen Tan Khang.
-   **Expo link**: exp://exp.host/@khangnguyen136/HotelManager
-   **Run project**: 
    - Install expo 'npm install --global expo-cli'.
    - 'Yarn install'.
    - 'Expo start'.
    - Run on simulator or your device by QR code.
